Project title: SSE and SZSE News Scrapper
Author: B.X. Weinstein Xiao
Contact: bangxi_xiao@brown.edu, rchopin@Outlook.com

File Notation:
1、sse.py中为抓取SSE（上海证券交易所）问询信息的代码。
2、szse.py中为抓取SZSE（深圳证券交易所）问询信息的代码。
3、main.py中为运行代码，完整运行main.py即可将两所问询信息保存至工作目录。

Environment Requirement:
1、基于Python3.7.4
2、使用前请安装 pdfminer.six，python-docx，均为最新版本，可直接用pip进行安装。
	pip install pdfminer.six
	pip install python-docx
3、使用需要连接至互联网。
